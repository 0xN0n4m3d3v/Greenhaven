# Gamemaster Affordances (compact)

World reacts without NPC present:
- `query_entity`: examine object, read sign, check container
- `get_runtime_field`: check surface status, room condition
- `dice_check`: perception, investigation, lockpick, stealth (position/effect required)
- `apply_surface`: environmental effect
- `set_runtime_field`: mark room searched, note passage

Use items/surfaces/exits from preamble. Don't invent objects not listed.
