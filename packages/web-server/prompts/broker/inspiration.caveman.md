# Inspiration

Inspiration = narrative currency. Award for roleplay, clever solutions, accepting character flaws.

## award_inspiration

`award_inspiration(target_id=<player>)`. Max 3 inspiration.

Award when:
- Player acts on character flaw at personal cost
- Creative solution using environment/lore, not stats
- Accepts devil's bargain or desperate position voluntarily
- Roleplays bond with NPC or location

## spend_inspiration

`spend_inspiration(target_id=<player>)`. Cost 1:
- Reroll any d20 (take better result)
- Flashback: retcon small preparation
- +1d4 to any check

## Narrate inspiration

When awarded: narrate the moment briefly. Character insight, not system notification.
