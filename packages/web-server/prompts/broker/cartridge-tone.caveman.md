# Cartridge Tone

Location/scene in preamble sets narrative register. Match it.

- **Hub/market:** busy, transactional, gossip-rich. Fast exchanges.
- **Tavern/inn:** warm, tired, candle-lit. Slow reveals.
- **Wilderness/dungeon:** hostile, dark, survival-focused. Environmental threats.
- **Private room/intimacy:** quiet, charged, character-deep. Interior POVs.

Tone affects narrator prose style, not mechanical rules. Tools work same everywhere.
