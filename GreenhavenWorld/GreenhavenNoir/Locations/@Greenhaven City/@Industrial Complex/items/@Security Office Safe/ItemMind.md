# @Security Office Safe

## Item Canon

A heavy combination safe in the security office. Contains the backup hard drive
from the CCTV system — the footage @Holst claimed was "looped" and overwritten.

## Description

The safe is old but functional — a four-digit mechanical combination lock.
@Holst claims he does not know the code. He is lying. But there is another
way: the safe's manufacturer default code is written on a maintenance sticker
on the back of @Holst's desk drawer.

## Usage

Open the safe to recover the backup hard drive. The default code is **7429** —
found by searching @Holst's desk. The hard drive contains unaltered security
footage from the night of the theft: Voss on camera, the precursors being
loaded onto a hand truck, and @Keller waiting by the loading dock.

## Materializes
- When the hero first examines @Security Office Safe:
  - Entity: `@Inspecting Security Office Safe`
  - Type: scene / item inspection
  - Scope: @Industrial Complex
  - Effect: the inspection scene opens — the hero sees the item in detail
    and discovers its secrets.

- When the hero opens the safe and recovers the hard drive:
  - Entity: `@CCTV Backup Footage`
  - Type: item / evidence
  - Scope: hero inventory
  - Effect: the hero now has definitive proof of Voss's involvement AND
    @Keller's role. This evidence can be used to pressure @Holst into a full
    confession. It also provides a clear image of Voss for the case board.

## Do Not Do Here

Do not let @Holst give up the code willingly. He must be pressured, or the
hero must find the sticker independently. The code is not guessable.
