# Action: Short Action Name

Коротко: какое действие игрока что-то открывает или меняет.

## Источник действия

- Где происходит: @Location Name
- С чем взаимодействует игрок: @Item or location

## Как игрок может это сделать

Обычным языком: "сдвинуть бочки", "прочитать письмо", "попросить Микку
перевести записку", "прислушаться у люка".

## Materializes

- Когда герой выполняет действие:
  - Entity: @Hidden location
  - Type: location / item / NPC / scene / quest / service / access / state
  - Scope: @Location Name
  - Effect: новый выход, новая память, новый доступ или новая награда.

## Как это должно прозвучать в игре

Короткое описание результата действия.

## Проверки

Нужна ли проверка навыка, предмет, помощь NPC или достаточно заявления игрока.
