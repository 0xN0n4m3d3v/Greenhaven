# @Scene Name

Коротко: какая сцена запускается в этой локации или у этого NPC, и какую
драматическую работу она делает.

## Где и когда

- Локация: @Location Name
- Участники: @NPC Name
- Если это сцена персонажа, владелец: @NPC Name
- Видимость: сразу / по разговору / после квеста / после действия
- Повторяемость: один раз / можно повторять / ежедневная рутина

## Завязка

Что видит игрок в первый момент сцены.

## Ход сцены

Опиши 2-5 возможных beats: начало, давление, выбор, последствие.

## Игровое состояние

Что может измениться: память, strings, квест, инвентарь, открытая локация,
состояние NPC или места.

## Materializes

Если сцена ничего не создает и не открывает, этот блок можно удалить.

- Когда сцена доходит до ...:
  - Entity: @New Entity Name
  - Type: location / item / NPC / scene / quest / service / access / state
  - Scope: @Location Name
  - Effect: ...

## Не делать

Что агент не должен выдумывать в этой сцене.
