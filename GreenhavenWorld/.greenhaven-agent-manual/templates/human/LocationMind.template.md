# @Location Name

Коротко: что это за место и почему игроку сюда интересно попасть.

## Канон места

- Родительская область: @City or region
- Видимость: открыта сразу / скрыта до действия / слух
- Стартовая локация: да / нет
- Тон: безопасно / тревожно / интимно / опасно / торжественно

## Описание для мастера

Опиши место обычной прозой: что видно первым, кто здесь бывает, какие звуки,
запахи, следы, настроение.

## Что здесь есть

- Персонажи: @NPC Name
- Предметы/объекты: @Item Name
- Сцены: @Scene Name
- Квестовые зацепки: Quest Name

## Выходы и скрытые пути

- Открытый путь: @Another location
- Скрытый путь: если игрок делает действие, открывается @Hidden location

## Правила локации

Что здесь нельзя забывать агенту: охрана, цены, социальные правила,
опасности, запреты, кто наблюдает за игроком.

## Картинки

Клади изображения в `images/`. Если картинки еще нет, напиши, какая нужна.
