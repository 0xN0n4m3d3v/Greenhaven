# @NPC Scene Name

Коротко: какая сцена принадлежит этому NPC. Это не общий квест и не сцена
локации, а персональная сцена персонажа: первая встреча, торговля, драка,
страх, интимность, рутина, предательство, забота или другой beat.

## Владелец сцены

- NPC: @NPC Name
- Где обычно применимо: @Location Name / любая локация / только рядом с целью
- Тип: первая встреча / бой / бегство / торговля / интимность / страх /
  предательство / забота / рутина / другое
- Приоритет: высокий / средний / низкий

## Когда включается

Когда эта сцена должна включаться.

Примеры:

- герой впервые замечает @NPC Name;
- начинается драка рядом с @NPC Name;
- игрок пытается обмануть при оплате;
- игрок делает взрослое предложение;
- NPC загнан в угол;
- NPC видит конкретного врага или предмет.
