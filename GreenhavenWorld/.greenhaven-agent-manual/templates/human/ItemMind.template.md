# @Item Name

Коротко: что это за предмет/объект и зачем он нужен в игре.

## Канон предмета

- Тип: переносимый предмет / декорация / контейнер / дверь / улика / валюта
- Где находится: @Location Name
- Можно взять: да / нет / только после условия
- Видимость: видно сразу / скрыто / заметно после проверки

## Описание

Как предмет выглядит, в каком состоянии, чем пахнет/звучит, почему выделяется.

## Использование

Что игрок может сделать: взять, открыть, сдвинуть, прочитать, купить, продать,
сломать, применить к другой сущности.

## Последствия

Что меняется после действия: открывается @Location Name, начинается квест,
появляется предмет в инвентаре, меняется память NPC.

## Materializes

Если предмет ничего не создает и не открывает, этот блок можно удалить.

- Когда герой делает ...:

  - Type: location / item / NPC / scene / quest / service / access / state
